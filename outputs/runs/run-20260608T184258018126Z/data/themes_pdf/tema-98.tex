Situacao: Julgado\\
Ramo: DIREITO TRIBUTÁRIO\\
Questao: Saber se incide contribuição previdenciária sobre férias de trabalhador avulso, verba de natureza indenizatória.\\
Tese: É excepcional a natureza indenizatória das férias de trabalhador avulso, a qual deve ser comprovada pela parte autora. Caso não o faça, se presume as goze anualmente, ocasião em que incide contribuição previdenciária.\\
